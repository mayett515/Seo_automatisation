# Gamification UX Language

## Sprache

Intern kann man von Spread, Boss-Level und Konkurrenten ausschalten sprechen. Im Kundenprodukt sollte es seriöser klingen.

| Intern | Kundenfreundlich |
|---|---|
| Spreaden | Gebiet ausbauen |
| Gegner ausschalten | Konkurrenz überholen |
| Boss-Level | schwieriger Hauptmarkt |
| Attacke | gezielter Ausbau |
| Erobern | Sichtbarkeit gewinnen |
| Süchtig machen | motivierender Fortschritt |

Dachau ist ein schwieriger Markt — aber wir sind jetzt im Spiel.
