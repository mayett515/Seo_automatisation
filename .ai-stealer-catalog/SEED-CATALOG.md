# Seed Repo Catalog

This folder contains the project-local seed catalog for the "A Good Artist Steals" workflow.

## Edit Policy

Do not casually rewrite the seed catalog during normal project work.

New sources, research notes, extracted architecture patterns, and implementation decisions belong in:

- `.ai-stealer-findings/`
- project planning docs
- ADRs
- `.ai-stealer-rules/` when the workflow itself changes

## Usage

Use this catalog as the starting map for finding proven solution shapes, module boundaries, data flows, APIs, state machines, and folder structures.

Dynamic research is expected. The seed catalog is stable; the findings are editable.
