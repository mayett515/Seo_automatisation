## Component Patterns To Steal

### Navbars And Menus
- Sticky header with scroll-hide/show behavior.
- Mega menu with multi-column dropdowns.
- Mobile hamburger with slide-in drawer.
- Transparent → solid background on scroll.
- Language switcher and dark mode toggle in nav.

### Hero Sections
- Full-screen hero with background image/video.
- Split hero: text left, image/form right.
- Gradient overlay with CTA button pair.
- Animated text/tagline cycling.
- Stats row below hero (customers served, years, projects).

### Cards And Grids
- Service cards: icon + title + description + link.
- Team/profile cards: photo + name + role + social links.
- Pricing cards: plan name, price, feature list, CTA.
- Blog/post cards: image + category tag + title + date + excerpt.
- Testimonial cards: quote + avatar + name + company.

### Sliders And Carousels
- Logo/tool carousel: auto-scrolling partner/client logos.
- Testimonial slider: quotes with navigation dots.
- Image gallery carousel with lightbox.
- Before/after comparison slider (cleaning results, renovation).
- Hero slider with multiple slides and custom transitions.

### Forms And CTAs
- Contact form: name, email, subject, message, submit.
- Newsletter signup: email input + subscribe button.
- Quote/booking request form with service selection.
- Floating CTA button (call now, WhatsApp, get quote).
- Cookie consent banner and GDPR notice.

### Hero Pattern-Katalog (Zum Stehlen)

**Full-Screen Moving Banner:**
- `<video autoplay muted loop>` as hero background with dark overlay + centered text + CTA buttons.
- Example CSS: `object-fit: cover; position: absolute; width: 100%; height: 100%;`
- Gatsby/Hero slider: multiple background slides with ken burns (slow zoom) effect.

**Sticky Header + Transparent → Solid:**
- Navbar: `position: fixed; background: transparent;` → `background: white; box-shadow` on scroll.
- JavaScript pattern: `window.addEventListener('scroll', () => { if(scrollY > 50) addClass('scrolled') })`
- CSS-only variant: `@supports (animation-timeline: scroll())` for scroll-driven header style changes.

**Split Hero (Text + Bild):**
- `.hero { display: grid; grid-template-columns: 1fr 1fr; }` — text left, image/video right.
- Mobile: stack vertically with text first.

**Gradient & Glassmorphism Hero:**
- `background: linear-gradient(135deg, #667eea 0%, #764ba2 100%)` — bold gradient hero.
- `backdrop-filter: blur(10px); background: rgba(255,255,255,0.1);` — glassmorphism nav on top of hero.

**Countdown / Launch Banner:**
- "Wir eröffnen bald" mit Countdown-Timer im Hero.
- Kombinierbar mit `countUp.js` für "120+ zufriedene Kunden" im Hero.

### Effekte Für Dienstleistungsseiten (Service Business Effects)

**"Mehr anzeigen" / Show More Patterns:**
- Expandable service cards: grid of cards → click "Mehr erfahren" → card expands inline with details.
- Accordion FAQ: click question → answer slides down (max-height transition).
- Read more text truncation: "Lorem ipsum… Mehr lesen" → full text reveals with smooth height animation.
- Filterable service grid: click category button → grid items filter with fade/scale transitions.

**Scroll-Reveal für Service-Sektionen:**
- Leistungen grid: each card fades up as you scroll down (AOS: `data-aos="fade-up"` with stagger delay).
- About section: image slides in from left, text from right.
- Testimonials: quotes appear one by one with `whileInView` (Motion) or Intersection Observer.
- Zahlen/Stats counter: numbers count up from 0 when section scrolls into view.

**Grid & Flexbox Layout-Tools:**
- CSS Grid: `grid-template-columns: repeat(auto-fill, minmax(300px, 1fr))` for responsive service cards.
- Flexbox: `flex-wrap: wrap` + `gap` for icon + text feature lists.
- Subgrid: align nested card content (title, description, button) across a row.
- Masonry layouts: Pinterest-style grid with varying heights using CSS columns or JS.
