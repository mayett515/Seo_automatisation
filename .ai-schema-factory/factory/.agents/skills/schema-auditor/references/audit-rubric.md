# Schema Audit Rubric

## Critical issues

- Missing `AGENTS.md`.
- Missing root router.
- Broken `SKILL.md` frontmatter.
- Empty critical files.
- Duplicate contradictory authority.
- Deep active rule nesting.
- Terminal leaves route downward.
- Private source material included in public repo.

## Medium issues

- Weak skill descriptions.
- Missing trigger phrases.
- Missing test prompts.
- Too many rules in one file.
- Missing anti-regression rules.
- Mixed Codex-specific and portable schema logic.

## Nice-to-have

- Pretty diagrams.
- Workflow docs.
- Cheat sheets.
- Hooks after real usage.
- Subagents for large research tasks.
