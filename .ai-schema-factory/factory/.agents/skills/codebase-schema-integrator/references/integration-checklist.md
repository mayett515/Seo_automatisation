# Codebase Integration Checklist

Before editing:

- [ ] Check `git status`.
- [ ] Inspect root files.
- [ ] Find existing agent instruction files.
- [ ] Identify package manager.
- [ ] Identify test/build/lint commands.
- [ ] Identify architecture folders.
- [ ] Propose plan.
- [ ] Wait for approval.

After editing:

- [ ] Show changed files.
- [ ] Show new tree.
- [ ] Provide rollback command.
- [ ] Provide first test prompt.
- [ ] Recommend commit message.
