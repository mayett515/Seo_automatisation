# Page Proposal Smoke Run Notes

Store only sanitized review notes for synthetic-fixture, credentialed Page Proposal smoke runs here.

Each note should contain:

- run id and UTC timestamp;
- provider/model;
- terminal status and stable failure/gate code;
- latency and usage/cost totals when supplied;
- draft proposal route and preview-version status when persistence succeeded;
- the object-storage `input_ref` key only;
- a short operator judgment on German/local plausibility, evidence honesty, and calibration needs.

Do not commit raw prompts, PageJson/provider output, evidence packets, provider bodies, secrets, competitor copy, customer data, or chain-of-thought.
